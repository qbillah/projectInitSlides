PROJECT INIT, is a small script for web developers who like to hack. This script helps you create and initialize files / directory structures for any web development project. *Composer is required for PHP Projects.
3 Types of projects to choose from:
1 - Plain HTML / CSS, 2 - HTML, CSS, and Javascript, 3 - PHP
  • Creates the project in the root directory.
  • Loads all necessary dependencies.
  • Adds a .gitignore file, and adds in all the dependencies to the file.
  • Adds a .json file for package management and dependencies.
  • Creates a manageable file structure for your project.
  • Creates files for you to get started with on your project.
